# Home Dashboard

The home page is the main entry point after login.

## Key Elements

### Header Bar
- Hamburger menu (top-left) - expands/collapses sidebar
- Semrush logo
- "Enterprise" link
- "More" dropdown
- User avatar (top-right)

### Global Search
- Large search bar: "Enter your task, website, or keyword"
- Accepts domains, keywords, or task descriptions
- Search icon button to submit

### Main CTA Card
- Headline: "Start optimizing online presence"
- Input field placeholder: "Enter a website"
- "Start now" button (teal/green) to submit the domain

### Footer Elements
- "Send feedback" link (bottom-left)
- Language selector: "English" with dropdown (bottom-center)
- Help button (orange circle with "?" - bottom-right)

## Sidebar Navigation (Toolkits)

Click hamburger menu to reveal full sidebar:

| Toolkit | Icon | Description |
|---------|------|-------------|
| Home | House | Main dashboard |
| SEO | Target | Search engine optimization tools |
| AI | Sparkles | AI-powered features (may show notification dot) |
| Content | Pencil | Content marketing tools |
| Traffic & Market | Chart | Traffic analysis and market research |
| Local | Pin | Local SEO tools |
| Social | Heart | Social media tools |
| Ad | Megaphone | Advertising/PPC tools |
| AI PR | Robot | AI public relations |
| Examples | Chart | Sample reports and data |
| Reports | Document | Custom reporting |
| App Center | Grid | Third-party integrations |

## Main Actions

### Analyze a domain
1. Type domain (e.g., "semrush.com") into the "Enter a website" input field
2. Click "Start now" button
3. Redirects to SEO Dashboard with domain metrics

### Navigate to tools
1. Click hamburger menu icon (top-left)
2. Select toolkit from sidebar (SEO, Content, etc.)
3. Each toolkit expands to show specific tools

### Access account
- Click user avatar (top-right) for account menu

## Tips
- The domain input accepts domains without http/https prefix
- After submitting, you'll be redirected to the SEO Dashboard
- Sidebar collapses to icons when hamburger menu is clicked again
- Look for notification dots on toolkit icons for new features
