# Home Dashboard

The home page is the main entry point after login.

## Key Elements
- "Start optimizing online presence" card with domain input field
- Input field placeholder: "Enter a website"
- "Start now" button (teal/green) to submit the domain
- Left sidebar with navigation icons to different tools

## Main Actions
- **Analyze a domain**: Type domain (e.g., "example.com") into the input field, click "Start now"
- **Navigate to tools**: Use left sidebar icons to access SEO, Analytics, etc.

## Tips
- The domain input accepts domains without http/https prefix
- After submitting, you'll be redirected to the domain overview page
